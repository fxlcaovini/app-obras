# 🏗️ App Obras

Sistema de gerenciamento de obras desenvolvido em **TypeScript**, criado para facilitar o controle administrativo e financeiro de empresas da construção civil.

## ✨ Funcionalidades

* 👥 Cadastro e gerenciamento de clientes
* 📋 Criação e controle de orçamentos
* 👷 Gestão de funcionários
* 🧱 Cadastro de materiais
* 💰 Controle financeiro da empresa

## 🚀 Tecnologias

* TypeScript

## ▶️ Como executar

```bash
git clone https://github.com/fxlcaovini/app-obras.git

cd app-obras

npm install

npm run dev
```
